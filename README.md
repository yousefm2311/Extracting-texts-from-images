# Extracting Texts From Images

A Flutter OCR project focused on extracting text from images using a Tesseract-style recognition workflow.

## Status

Public portfolio candidate.

## Features

- Image-to-text extraction concept
- OCR-focused Flutter workflow
- Mobile-first user interface foundation
- Ready for camera/gallery and result-screen documentation

## Tech Stack

- Flutter
- Dart
- OCR
- Tesseract concept

## Getting Started

```bash
git clone https://github.com/yousefm2311/Extracting-texts-from-images.git
cd Extracting-texts-from-images
flutter pub get
flutter run
```
